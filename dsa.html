1. Longest Common Subsequence (LCS)

Given two strings, find the length of the longest subsequence present in both.

Delulu's Memo Hack ðŸ§ 

Match or Shift: If text1[i] == text2[j], it's a match (+1). If not, explore two possibilities: skip char in text1 OR skip char in text2. Result is max(skip1, skip2). State: (i, j).

View Raja's Implementation 
class Solution: 
    def longestCommonSubsequence(self, text1: str, text2: str) -> int: 
        l1 , l2 = len(text1) , len(text2) 
        memo = {} 
        def search(i , j ): 
            if i >= l1 or j>=l2: 
                return 0 
            if (i , j ) in memo: 
                return memo[(i , j)] 
            res = 0 
            if text1[i] == text2[j]: 
                res = max( res , search(i+1 , j+1)+1) 
            l = search(i+1 , j) 
            r = search(i , j+1) 
            res = max(l , r , res) 
            memo[(i , j)] = res 
            return res 
        return search(0 , 0)

3. Partition Equal Subset Sum

Determine if an array can be partitioned into two subsets such that the sum of elements in both subsets is equal.

Delulu's Memo Hack ðŸ§ 

Target Sum: If total sum is odd, impossible. Otherwise, target = total sum / 2. Decision: for each number, either include it in the sum or don't. Memoize the (index, current_sum).

View Raja's Implementation 
from functools import lru_cache
class Solution:
    def canPartition(self, nums: List[int]) -> bool:
        le = len(nums)
        @lru_cache()
        def search(i , l , r):
            if i >= le:
                return True if l==r else False
            cur= nums[i]
            l1 , l2 = search(i+1 ,l+cur , r) , search(i+1, l , r+cur)
            return l1 or l2
        return search(0 , 0 ,0 )

4. Longest Increasing Subsequence

Find the length of the longest strictly increasing subsequence in an array.

Delulu's Memo Hack ðŸ§ 

Compare with Prev: At each element, if it's greater than the previous included element, you can include it (+1) or skip it. If not, you must skip it. State: (current_index, value_of_prev).

View Raja's Implementation 
from functools import lru_cache
class Solution:
    def lengthOfLIS(self, nums: List[int]) -> int:
        l = len(nums)
        @lru_cache(None)
        def search(n , prev):
            if n >= l:
                return 0
            cur = nums[n]
            res = 0
            if cur > prev:
                res = search(n+1, cur) + 1
            res = max(search(n+1, prev), res)
            return res
        return search(0, float("-inf"))
