1. Partition Equal Subset Sum

Determine if an array can be partitioned into two subsets with equal sums.

Delulu's Memo Hack ðŸ§ 

2-Way Branching: For each element, it's either in Subset A or B. If sum(A) == sum(B) at the end, return True. Shortcut: total_sum must be even; search for total_sum/2.

View Raja's Implementation 
from functools import lru_cache
class Solution:
    def canPartition(self, nums: List[int]) -> bool:
        le = len(nums)
        @lru_cache()
        def search(i , l , r):
            if i >= le:
                return True if l==r else False
            cur= nums[i]
            l1 , l2 = search(i+1 ,l+cur , r) , search(i+1, l , r+cur)
            return l1 or l2
        return search(0 , 0 ,0 )

2. Longest Increasing Subsequence (LIS)

Find the length of the longest strictly increasing subsequence in an array.

Delulu's Memo Hack ðŸ§ 

Include/Exclude: At each step, if cur > prev, you can choose to include it (+1) or skip it. Maximize the result of both branches. State: (current_index, value_of_last_included).

View Raja's Implementation 
from functools import lru_cache
class Solution:
    def lengthOfLIS(self, nums: List[int]) -> int:
        l = len(nums)
        @lru_cache(None)
        def search(n , prev):
            if n >= l:
                return 0
            cur = nums[n]
            res = 0
            if cur > prev:
                res = search(n+1, cur) + 1
            res = max(search(n+1, prev), res)
            return res
        return search(0, float("-inf"))
