01. THE INGESTION (Insert)

Storing data in BSON (Binary JSON) format.

QUERY_BUFFER
db.students.insertOne({
  name: "Raja Bolla",
  role: "Billionaire Founder",
  skills: ["Python", "JS", "AI"]
})
{ "acknowledged" : true, "insertedId" : ObjectId("65d0a1b...") }

02. THE DISCOVERY (Find)

Filtering documents with specific criteria.

QUERY_BUFFER
db.students.find({
  skills: "AI"
}).pretty()
[
  { "_id": "...", "name": "Raja Bolla", "skills": ["Python", "JS", "AI"] }
]

03. THE PIPELINE (Aggregate)

The power of MongoDB. Transform, group, and calculate data in stages.

QUERY_BUFFER
db.students.aggregate([
  { $match: { role: "Billionaire Founder" } },
  { $unwind: "$skills" },
  { $group: { _id: "$skills", count: { $sum: 1 } } }
])
{ "_id": "AI", "count": 1 }
{ "_id": "Python", "count": 1 }
{ "_id": "JS", "count": 1 }

04. THE EVOLUTION (Update)

Using atomic operators like $set or $push to modify existing docs.

QUERY_BUFFER
db.students.updateOne(
  { name: "Raja Bolla" },
  { $push: { skills: "MongoDB Master" } }
)
{ "matchedCount" : 1, "modifiedCount" : 1 }
