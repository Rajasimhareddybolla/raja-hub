Module 01 // Advanced CRUD & Comparison

Comparison Operators ($gt, $in, $ne)

Finding documents based on ranges or specific lists.

QUERY_BUFFER
db.products.find({
  price: { $gt: 500, $lt: 2000 },
  category: { $in: ["Electronics", "Laptops"] },
  stock: { $ne: 0 }
})
[
  { "name": "M3 MacBook Air", "price": 1099, "category": "Laptops", "stock": 15 }
]
Module 02 // Array Power ($all, $elemMatch, $size)

Array Element Matching

Querying documents that contain arrays matching specific complex conditions.

QUERY_BUFFER
db.users.find({
  grades: {
    $elemMatch: { score: { $gt: 90 }, subject: "ML" }
  }
})
[
  { "user": "Raja", "grades": [{ "subject": "ML", "score": 98 }] }
]
Module 03 // Atomic Update Operators ($inc, $rename, $unset)

Field Transformation

Modifying document structure without rewriting the whole object.

QUERY_BUFFER
db.students.updateMany(
  { year: 2025 },
  {
    $inc: { credits: 4 },
    $rename: { "old_email": "email" },
    $set: { status: "PROMOTED" }
  }
)
{ "matchedCount" : 42, "modifiedCount" : 42 }
Module 04 // Aggregation Mastery ($lookup, $project, $sort)

The JOIN Equivalent ($lookup)

Performing left-outer joins between collections.

QUERY_BUFFER
db.orders.aggregate([
  {
    $lookup: {
      from: "inventory",
      localField: "item",
      foreignField: "sku",
      as: "inventory_details"
    }
  },
  { $unwind: "$inventory_details" },
  { $project: { order_id: 1, "inventory_details.stock": 1 } }
])
{ "_id": 1, "order_id": "ABC", "inventory_details": { "stock": 450 } }
